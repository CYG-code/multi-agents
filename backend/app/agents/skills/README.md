# Agent Skills Directory

This directory stores role-specific SKILL packages for backend agents.

Naming convention:
- one folder per role, snake_case
- examples: `devil_advocate`, `facilitator`, `summarizer`, `encourager`

Recommended package layout:
- `SKILL.md` (required)
- `references/` (optional)
- `assets/` (optional)
- `scripts/` (optional)
- `agents/` (optional)
