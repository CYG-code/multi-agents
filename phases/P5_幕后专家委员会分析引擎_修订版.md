# P5 阶段：幕后专家委员会（分析引擎）开发（修订版）

**所属项目：** 多智能体辅助在线协作学习平台 v1.2  
**阶段编号：** P5 / 8  
**预计周期：** 1-2 周  
**前置依赖：** P4（AgentWorker、任务队列、5 个角色 Agent、基础版 Committee）  
**修订原则：** 后台专家只诊断；ChiefDispatcher 只做调度决策；前台 Role Agent 才生成学生可见发言。

---

## 一、阶段目标

在 P4 阶段统计规则委员会的基础上，引入真正的 AI 分析专家机制：三位 Analyst 并发调用 LLM 输出结构化诊断报告，ChiefDispatcher 综合三份报告后输出**单一调度决策**。

P5 不再简单“替换”P4 规则委员会，而是形成：

```text
规则指标负责可计算事实；
LLM Analyst 负责解释与诊断；
ChiefDispatcher 负责是否干预与派谁；
AgentWorker 负责频率、冷却、互斥锁等执行门控；
Role Agent 负责生成学生可见发言。
```

**完成标志：**

1. 触发一次定时分析，后台日志显示三位 Analyst 并发完成。
2. 教师端出现包含三份专家诊断、规则指标、ChiefDispatcher 决策的分析快照。
3. ChiefDispatcher 能输出 `should_intervene=false`，系统不会为了分析而强行干预。
4. 若需要干预，每次委员会分析最多生成 1 个前台 Agent 任务。
5. C 类 @Agent 主动召唤不等待专家分析，能直接进入队列并响应。
6. `agent_interventions` 能记录从 queued 到 executed/failed/dropped 的完整状态，并能回写最终 `message_id`。

## 二、核心设计约束（必须遵守）

### 2.1 后台专家只诊断，不直接发言

三位 Analyst 的输出面向系统内部和教师端，不直接展示给学生。它们不能生成学生可见话术，也不能直接决定派哪个前台角色。

### 2.2 ChiefDispatcher 每次最多派 1 个前台 Agent

P5 不采用 `interventions: []` 多任务列表结构。ChiefDispatcher 必须输出一个单一 `DispatchDecision`：

```json
{
  "should_intervene": true,
  "selected_agent_role": "devil_advocate",
  "priority": 1,
  "intervention_goal": "促进反例检验，避免浅层共识",
  "reason": "小组已形成初步一致观点，但缺少反例、证据和边界条件检验。",
  "strategy": "提出一个温和的反向假设，要求小组检验当前方案在特殊条件下是否仍成立。",
  "target_dimension": "cognitive",
  "evidence": ["最近多条消息为直接同意", "当前方案尚未讨论潜在风险"]
}
```

如果当前不适合干预，必须允许：

```json
{
  "should_intervene": false,
  "reason": "小组正在自然推进讨论，当前无需打断。",
  "observe_next": "继续观察观点多样性和低参与成员情况。"
}
```

### 2.3 默认不干预

ChiefDispatcher 的默认策略是 `should_intervene=false`。只有当 AI 干预明显有助于当前协作推进时，才允许派遣前台 Agent。

### 2.4 不保留第二套 B 类规则调度

B 类 AI 驱动调度统一由 ChiefDispatcher 完成，不再另写 `check_b_class_triggers()` 进行额外入队，避免重复干预。

### 2.5 所有队列任务必须可追溯

委员会触发、主动 @、沉默触发、垄断触发都必须写入 `agent_interventions`，并在队列任务中携带：

```json
{
  "trigger_type": "committee",
  "snapshot_id": "...",
  "intervention_id": "...",
  "reason": "...",
  "strategy": "..."
}
```

## 六、Metrics 规则指标层

**文件：** `backend/app/analysis/metrics.py`

规则指标不替代 Analyst，但作为可计算事实、LLM 输入和失败 fallback。

```python
from __future__ import annotations

from collections import Counter


def calculate_participation_scores(messages: list[dict], members: list[dict]) -> dict:
    """
    计算各成员参与度。
    必须包含沉默成员，不能只统计发过言的人。
    返回 {user_id: {display_name, score, message_count}}。
    """
    member_map = {str(m["id"]): m.get("display_name", "") for m in members}
    counts = {uid: 0 for uid in member_map}

    for msg in messages:
        if msg.get("sender_type") != "student":
            continue
        sender_id = str(msg.get("sender_id") or "")
        if sender_id in counts:
            counts[sender_id] += 1

    total = sum(counts.values())
    return {
        uid: {
            "display_name": member_map.get(uid, ""),
            "score": (count / total) if total else 0.0,
            "message_count": count,
        }
        for uid, count in counts.items()
    }


def calculate_balance_score(participation_scores: dict) -> float:
    """
    协作均衡性 = 1 - 基尼系数。
    越接近 1 越均衡。
    """
    if not participation_scores:
        return 0.5

    values = sorted(float(v.get("score", 0.0)) for v in participation_scores.values())
    n = len(values)
    total = sum(values)
    if n == 0 or total == 0:
        return 0.5

    cumulative = sum((i + 1) * v for i, v in enumerate(values))
    gini = (2 * cumulative) / (n * total) - (n + 1) / n
    return max(0.0, min(1.0 - abs(gini), 1.0))


def extract_message_stats(messages: list[dict], members: list[dict]) -> dict:
    student_msgs = [m for m in messages if m.get("sender_type") == "student"]
    agent_msgs = [m for m in messages if m.get("sender_type") == "agent"]
    sender_counts = Counter(m.get("sender_id") for m in student_msgs if m.get("sender_id"))

    silent_member_ids = [
        str(m["id"])
        for m in members
        if str(m["id"]) not in sender_counts
    ]

    return {
        "total_count": len(messages),
        "student_count": len(student_msgs),
        "agent_count": len(agent_msgs),
        "member_count": len(members),
        "unique_student_participants": len(sender_counts),
        "silent_member_ids": silent_member_ids,
        "avg_student_message_length": (
            sum(len(m.get("content") or "") for m in student_msgs) / max(len(student_msgs), 1)
        ),
        "sender_counts": dict(sender_counts),
    }
```

## 八、JSON 解析工具

不要在每个 Analyst 内重复写 Markdown 代码块清洗逻辑。

**文件：** `backend/app/background_experts/json_utils.py`

```python
from __future__ import annotations

import json


def extract_json_object(raw: str) -> dict:
    text = (raw or "").strip()

    if text.startswith("```"):
        text = text.strip("`").strip()
        if text.startswith("json"):
            text = text[4:].strip()

    start = text.find("{")
    end = text.rfind("}")
    if start == -1 or end == -1 or end <= start:
        raise ValueError("No JSON object found in LLM output")

    return json.loads(text[start : end + 1])
```

### 10.2 CognitiveAnalyst 示例

```python
import json
from app.agents.settings import get_agent_settings
from app.analysis.metrics import extract_message_stats
from app.background_experts.base import BaseExpert
from app.background_experts.schemas import CognitiveAnalysisResult


class CognitiveAnalyst(BaseExpert):
    PROMPT_FILE = "cognitive_analyst_prompt.txt"
    MAX_TOKENS = 1024

    def __init__(self):
        cfg = get_agent_settings()
        super().__init__(model=cfg.models.cognitive_analyst.model_version)
        self.token_budget = cfg.models.cognitive_analyst.history_token_budget

    async def analyze(self, context: dict, messages: list[dict], rule_metrics: dict) -> CognitiveAnalysisResult:
        try:
            system_prompt = self._prompt_template.format(
                task_requirements=context.get("task_description", ""),
                current_phase=context.get("current_phase", ""),
                phase_goal=context.get("phase_goal", ""),
                rule_metrics=json.dumps(rule_metrics, ensure_ascii=False),
                message_count=len(messages),
                chat_history=self._format_history(messages),
            )
            result = await self._call_json(system_prompt)
            return CognitiveAnalysisResult.model_validate(result)
        except Exception as exc:
            return CognitiveAnalysisResult(
                diversity_score=0.5,
                progress_score=0.5,
                cognitive_report={
                    "diagnosis": "认知分析暂时不可用，已使用默认值。",
                    "error": str(exc),
                    "recommended_intervention_type": "none",
                    "severity": 0.0,
                    "evidence": [],
                },
            )

    def _format_history(self, messages: list[dict]) -> str:
        lines = []
        total_chars = 0
        char_budget = self.token_budget
        for msg in reversed(messages):
            line = f"[{msg.get('display_name', '?')}] {msg.get('content', '')}"
            if total_chars + len(line) > char_budget:
                lines.insert(0, "（以下为按 token budget 截断后的近期消息）")
                break
            lines.insert(0, line)
            total_chars += len(line)
        return "\n".join(lines)
```

## 十二、committee.py 编排层

**文件：** `backend/app/agents/committee.py`

```python
from __future__ import annotations

import asyncio
import time
import uuid
from sqlalchemy import update

from app.agents.context_builder import get_recent_messages, get_room_context, get_room_members
from app.agents.queue import enqueue_task
from app.analysis.metrics import (
    calculate_balance_score,
    calculate_participation_scores,
    extract_message_stats,
)
from app.background_experts.cognitive_analyst import CognitiveAnalyst
from app.background_experts.emotional_analyst import EmotionalAnalyst
from app.background_experts.interaction_analyst import InteractionAnalyst
from app.background_experts.chief_dispatcher import ChiefDispatcher
from app.db.redis_client import get_redis_client
from app.db.session import AsyncSessionLocal
from app.models.analysis import AnalysisSnapshot, AgentIntervention, InterventionStatus, TriggeredBy


cognitive_analyst = CognitiveAnalyst()
emotional_analyst = EmotionalAnalyst()
interaction_analyst = InteractionAnalyst()
chief_dispatcher = ChiefDispatcher()


class ExpertCommittee:
    async def analyze_and_dispatch(
        self,
        room_id: str,
        triggered_by: TriggeredBy = TriggeredBy.timer,
    ) -> str | None:
        redis_client = get_redis_client()
        lock_key = f"analysis_running:{room_id}"
        lock_value = str(uuid.uuid4())

        acquired = await redis_client.set(lock_key, lock_value, nx=True, ex=180)
        if not acquired:
            print(f"[Committee] room={room_id} analysis already running, skip")
            return None

        try:
            return await self._analyze_and_dispatch_locked(room_id, triggered_by)
        finally:
            current = await redis_client.get(lock_key)
            if current == lock_value:
                await redis_client.delete(lock_key)

    async def _analyze_and_dispatch_locked(
        self,
        room_id: str,
        triggered_by: TriggeredBy,
    ) -> str | None:
        messages = await get_recent_messages(room_id, limit=50)
        if not messages:
            return None

        context = await get_room_context(room_id)
        members = context.get("members") or await get_room_members(room_id)

        seq_nums = [m.get("seq_num") for m in messages if m.get("seq_num") is not None]
        msg_seq_from = min(seq_nums) if seq_nums else 0
        msg_seq_to = max(seq_nums) if seq_nums else 0

        participation = calculate_participation_scores(messages, members)
        balance = calculate_balance_score(participation)
        message_stats = extract_message_stats(messages, members)
        rule_metrics = {
            "participation_scores": participation,
            "balance_score": balance,
            "message_stats": message_stats,
        }

        print(f"[Committee] 开始并发分析 room={room_id}, messages={len(messages)}")

        cognitive_result, emotional_result, interaction_result = await asyncio.gather(
            cognitive_analyst.analyze(context, messages, rule_metrics),
            emotional_analyst.analyze(context, messages, rule_metrics),
            interaction_analyst.analyze(context, messages, rule_metrics),
        )

        cognitive_dict = cognitive_result.model_dump()
        emotional_dict = emotional_result.model_dump()
        interaction_dict = interaction_result.model_dump()

        snapshot_id = uuid.uuid4()
        async with AsyncSessionLocal() as db:
            snapshot = AnalysisSnapshot(
                id=snapshot_id,
                room_id=room_id,
                message_seq_from=msg_seq_from,
                message_seq_to=msg_seq_to,
                analyzed_message_count=len(messages),
                analysis_context={
                    "current_phase": context.get("current_phase"),
                    "phase_goal": context.get("phase_goal"),
                    "elapsed_minutes": context.get("elapsed_minutes"),
                    "recent_interventions": context.get("recent_interventions", []),
                },
                rule_metrics=rule_metrics,
                participation_scores=participation,
                diversity_score=cognitive_dict.get("diversity_score"),
                balance_score=interaction_dict.get("balance_score", balance),
                progress_score=cognitive_dict.get("progress_score"),
                emotion_flags=emotional_dict.get("emotion_flags"),
                cognitive_report=cognitive_dict.get("cognitive_report"),
                emotional_report=emotional_dict.get("emotional_report"),
                interaction_report=interaction_dict.get("interaction_report"),
                triggered_by=triggered_by,
            )
            db.add(snapshot)
            await db.commit()

        decision = await chief_dispatcher.dispatch(
            context=context,
            rule_metrics=rule_metrics,
            cognitive_result=cognitive_dict,
            emotional_result=emotional_dict,
            interaction_result=interaction_dict,
        )
        decision_dict = decision.model_dump()

        intervention_id = None
        async with AsyncSessionLocal() as db:
            await db.execute(
                update(AnalysisSnapshot)
                .where(AnalysisSnapshot.id == snapshot_id)
                .values(
                    dispatcher_summary=decision.dispatcher_summary,
                    dispatcher_decision=decision_dict,
                    should_intervene=decision.should_intervene,
                    selected_agent_role=decision.selected_agent_role,
                    selected_strategy=decision.strategy,
                )
            )

            if decision.should_intervene and decision.selected_agent_role:
                intervention_id = uuid.uuid4()
                intervention = AgentIntervention(
                    id=intervention_id,
                    room_id=room_id,
                    snapshot_id=snapshot_id,
                    trigger_type="committee",
                    agent_role=decision.selected_agent_role,
                    reason=decision.reason,
                    strategy=decision.strategy,
                    priority=decision.priority or 2,
                    status=InterventionStatus.queued,
                )
                db.add(intervention)

            await db.commit()

        if decision.should_intervene and decision.selected_agent_role and intervention_id:
            await enqueue_task(
                room_id,
                {
                    "room_id": room_id,
                    "agent_role": decision.selected_agent_role,
                    "reason": decision.reason,
                    "strategy": decision.strategy or "",
                    "priority": decision.priority or 2,
                    "trigger_type": "committee",
                    "snapshot_id": str(snapshot_id),
                    "intervention_id": str(intervention_id),
                    "triggered_at": time.time(),
                },
            )

        await self._push_to_teachers(room_id, str(snapshot_id))
        print(f"[Committee] 分析完成 room={room_id}, should_intervene={decision.should_intervene}")
        return str(snapshot_id)

    async def _push_to_teachers(self, room_id: str, snapshot_id: str) -> None:
        from app.websocket.manager import manager

        async with AsyncSessionLocal() as db:
            snapshot = await db.get(AnalysisSnapshot, snapshot_id)

        if not snapshot:
            return

        await manager.broadcast_to_teachers(
            room_id,
            {
                "type": "analysis:update",
                "snapshot": serialize_snapshot(snapshot),
            },
        )


expert_committee = ExpertCommittee()

    db: AsyncSession = Depends(get_db),
):
    result = await db.execute(
        select(AnalysisSnapshot)
        .where(AnalysisSnapshot.room_id == room_id)
        .order_by(AnalysisSnapshot.created_at.desc())
        .limit(limit)
    )
    snapshots = result.scalars().all()
    return {"snapshots": [serialize_snapshot(s) for s in snapshots]}


@analytics_router.get("/rooms/{room_id}/interventions")
async def get_interventions(
    room_id: UUID,
    limit: int = Query(50, le=100),
    current_user: User = Depends(require_teacher),
    db: AsyncSession = Depends(get_db),
):
    result = await db.execute(
        select(AgentIntervention)
        .where(AgentIntervention.room_id == room_id)
        .order_by(AgentIntervention.created_at.desc())
        .limit(limit)
    )
    interventions = result.scalars().all()
    return {"interventions": [serialize_intervention(i) for i in interventions]}


@analytics_router.get("/rooms/{room_id}/summary")
async def get_summary(
    room_id: UUID,
    current_user: User = Depends(require_teacher),
    db: AsyncSession = Depends(get_db),
):
    latest = await db.execute(
        select(AnalysisSnapshot)
        .where(AnalysisSnapshot.room_id == room_id)
        .order_by(AnalysisSnapshot.created_at.desc())
        .limit(1)
    )
    snapshot = latest.scalar_one_or_none()

    if not snapshot:
        return {"message": "暂无分析数据"}

    return serialize_snapshot(snapshot)
```

## 十八、防重复触发机制

### 18.1 A 类规则触发

```text
trigger_lock:{room_id}:monopoly    TTL = 60s
trigger_lock:{room_id}:silence     TTL = silence_threshold_seconds + 60s
```

### 18.2 专家委员会分析锁

```text
analysis_running:{room_id}         TTL = 180s
```

该锁必须在 `ExpertCommittee.analyze_and_dispatch()` 开头实现，防止 scheduler、debug API、手动测试同时触发重复分析。

### 18.3 执行层已有门控

AgentWorker 继续保留：

```text
1. 功能开关检查
2. 每房间每小时干预上限
3. 角色冷却 cooldown:{room_id}:{agent_role}
4. 房间互斥锁 room:{room_id}:agent_lock
5. 锁冲突时延迟重入队列
```

### 18.4 Dispatcher 层防重复

ChiefDispatcher 必须看到 `recent_interventions`，并遵守：

```text
避免连续派遣同一角色。
如果最近已经有相同角色干预，除非问题明显加重，否则不重复派遣。
```

## 二十二、实施风险提醒

1. **不要让 Analyst 直接派角色。** 角色选择只能由 ChiefDispatcher 完成。
2. **不要一次派多个 Agent。** 每轮委员会分析最多一个前台 Agent。
3. **不要删除 P4 规则指标。** 规则指标是可解释性、fallback 和教师端展示的基础。
4. **不要只记录 summary。** 必须保存完整 `dispatcher_decision`。
5. **不要忽略沉默成员。** participation_scores 必须包含所有成员。
6. **不要硬编码 current_phase。** 阶段错误会直接导致后台专家误判。
7. **不要直接返回 SQLAlchemy 对象。** Analytics API 必须显式序列化。
8. **不要保留第二套 B 类调度。** `check_b_class_triggers()` 应删除或不进入主流程。
9. **不要忘记 intervention_id。** 否则队列任务、干预记录、最终消息无法闭环。
10. **不要让情绪分析变成心理诊断。** 只能描述可观察语言信号。
