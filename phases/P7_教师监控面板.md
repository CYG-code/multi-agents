# P7 阶段：教师监控面板

**所属项目：** 多智能体辅助在线协作学习平台 v1.2  
**阶段编号：** P7 / 8  
**预计周期：** 1 周  
**前置依赖：** P5（分析快照 API、WebSocket 教师推送）、P6（界面风格体系）

---

## 一、阶段目标

基于已有分析数据，实现教师端可视化监控面板，展示协作实时状态、历史分析趋势与 AI 干预日志，支持实时接收 `analysis:update` WebSocket 推送并自动刷新图表。

**完成标志：** 教师账号打开监控面板，选择活跃房间，实时看到参与度图表、5 维度雷达图和 AI 干预日志，学生每次 AI 发言后数据自动更新。

---

## 二、开发方案总览

```
P7 分为 4 个子模块：

  A. 教师端路由与鉴权
  B. 核心图表组件（Chart.js）：参与度 / 雷达图 / 时间轴
  C. 干预记录列表 + 专家报告详情
  D. WebSocket 实时数据接入 + 消息监控流
```

### 步骤 2：TeacherDashboard 主视图

**文件：** `frontend/src/views/TeacherDashboard.vue`

```vue
<script setup>
import { ref, computed, watch, onMounted, onUnmounted } from 'vue'
import { useAuthStore } from '@/stores/auth'
import { useWebSocket } from '@/composables/useWebSocket'
import { analyticsApi } from '@/services/analyticsApi'
import ParticipationChart from '@/components/dashboard/ParticipationChart.vue'
import RadarChart from '@/components/dashboard/RadarChart.vue'
import TrendChart from '@/components/dashboard/TrendChart.vue'
import InterventionLog from '@/components/dashboard/InterventionLog.vue'
import MessageMonitor from '@/components/dashboard/MessageMonitor.vue'

const authStore = useAuthStore()
const rooms = ref([])
const selectedRoomId = ref(null)
const snapshots = ref([])       // 历史分析快照列表
const interventions = ref([])   // 干预记录
const latestSnapshot = ref(null) // 最新快照（实时更新）
const messages = ref([])        // 房间消息（只读监控）

// 当前房间的 WebSocket 连接（教师身份）
let wsManager = null

onMounted(async () => {
  // 加载房间列表
  const result = await roomApi.getRooms({ status: 'active' })
  rooms.value = result
  
  // 默认选择第一个房间
  if (rooms.value.length > 0) {
    selectRoom(rooms.value[0].id)
  }
})

async function selectRoom(roomId) {
  selectedRoomId.value = roomId
  
  // 加载历史数据
  const [snapshotData, interventionData] = await Promise.all([
    analyticsApi.getSnapshots(roomId),
    analyticsApi.getInterventions(roomId)
  ])
  snapshots.value = snapshotData.snapshots
  interventions.value = interventionData.interventions
  latestSnapshot.value = snapshots.value[0] || null
  
  // 连接 WebSocket（教师身份）
  connectTeacherWS(roomId)
}

function connectTeacherWS(roomId) {
  wsManager?.disconnect()
  wsManager = useWebSocket(roomId)
  wsManager.connect()
  
  // 监听实时分析更新
  wsManager.on('analysis:update', (data) => {
    latestSnapshot.value = data.snapshot
    snapshots.value.unshift(data.snapshot)  // 插入最新快照
    
    // 滚动图表动画更新由 Chart.js 响应式 props 触发
  })
  
  // 监听聊天消息（只读监控）
  wsManager.on('chat:new_message', (data) => {
    messages.value.push(data)
    if (messages.value.length > 100) messages.value.shift()  // 最多保留100条
  })
  
  // 监听 AI 干预（实时添加到干预记录）
  wsManager.on('agent:stream_end', (data) => {
    if (data.status === 'ok') {
      // 刷新干预记录
      analyticsApi.getInterventions(roomId, { limit: 5 }).then(r => {
        // 追加最新干预记录（避免重复）
        const newIds = new Set(interventions.value.map(i => i.id))
        r.interventions.forEach(i => {
          if (!newIds.has(i.id)) interventions.value.unshift(i)
        })
      })
    }
  })
}

onUnmounted(() => {
  wsManager?.disconnect()
})

// 计算图表数据
const radarData = computed(() => ({
  labels: ['参与度', '论点多样性', '协作均衡', '任务进展', '情绪健康'],
  datasets: [{
    label: '当前协作状态',
    data: latestSnapshot.value ? [
      Math.min(Object.keys(latestSnapshot.value.participation_scores || {}).length / 5, 1) * 100,
      (latestSnapshot.value.diversity_score || 0) * 100,
      (latestSnapshot.value.balance_score || 0) * 100,
      (latestSnapshot.value.progress_score || 0) * 100,
      getEmotionHealthScore(latestSnapshot.value.emotion_flags) * 100
    ] : [0, 0, 0, 0, 0],
    backgroundColor: 'rgba(59, 130, 246, 0.2)',
    borderColor: 'rgb(59, 130, 246)',
    borderWidth: 2
  }]
}))

function getEmotionHealthScore(flags) {
  if (!flags) return 0.5
  const penalty = (flags.conflict ? 0.4 : 0) + (flags.passive ? 0.3 : 0) + (flags.anxious ? 0.2 : 0)
  return Math.max(0, 1 - penalty)
}
</script>

<template>
  <div class="min-h-screen bg-gray-50 flex flex-col">
    <!-- 顶部导航 -->
    <nav class="bg-white border-b border-gray-200 px-6 py-3 flex items-center gap-4">
      <h1 class="text-lg font-bold text-gray-800">教师监控面板</h1>
      
      <!-- 房间选择器 -->
      <select
        v-model="selectedRoomId"
        @change="selectRoom(selectedRoomId)"
        class="ml-4 px-3 py-1.5 border border-gray-300 rounded-lg text-sm"
      >
        <option v-for="room in rooms" :key="room.id" :value="room.id">
          {{ room.name }}
        </option>
      </select>
      
      <!-- 实时状态指示 -->
      <div class="flex items-center gap-1.5 ml-auto">
        <div class="w-2 h-2 rounded-full bg-green-400 animate-pulse" />
        <span class="text-xs text-gray-500">实时监控中</span>
      </div>
    </nav>
    
    <div class="flex-1 p-6 space-y-6 max-w-7xl mx-auto w-full">
      <!-- 指标卡区域 -->
      <MetricsCards :snapshot="latestSnapshot" />
      
      <!-- 图表区域（两列） -->
      <div class="grid grid-cols-2 gap-6">
        <ParticipationChart :snapshots="snapshots" />
        <RadarChart :data="radarData" />
      </div>
      
      <!-- 时间轴趋势图（全宽） -->
      <TrendChart :snapshots="snapshots" />
      
      <!-- 底部（两列） -->
      <div class="grid grid-cols-2 gap-6">
        <InterventionLog :interventions="interventions" />
        <MessageMonitor :messages="messages" />
      </div>
    </div>
  </div>
</template>
```

```vue
<script setup>
import { computed } from 'vue'

const props = defineProps({ snapshot: Object })

const cards = computed(() => [
  {

    value: props.snapshot ? `${(props.snapshot.diversity_score * 100).toFixed(0)}%` : '—',
    icon: '💡',

  {
    title: '协作均衡性',
    value: props.snapshot ? `${(props.snapshot.balance_score * 100).toFixed(0)}%` : '—',
    icon: '⚖️',
    color: getScoreColor(props.snapshot?.balance_score),
    desc: props.snapshot?.balance_score < 0.4 ? '⚠ 存在垄断' : '均衡'
  },
  {
    title: '任务进展',
    value: props.snapshot ? `${(props.snapshot.progress_score * 100).toFixed(0)}%` : '—',
    icon: '📋',
    color: getScoreColor(props.snapshot?.progress_score),
    desc: '与任务目标对比'
  },
  {
    title: '情绪状态',
    value: getEmotionLabel(props.snapshot?.emotion_flags),
    icon: '😊',
    color: hasNegativeEmotion(props.snapshot?.emotion_flags) ? 'red' : 'green',
    desc: getEmotionDesc(props.snapshot?.emotion_flags)
  }
])

function getScoreColor(score) {
  if (score === undefined || score === null) return 'gray'
  if (score >= 0.7) return 'green'
  if (score >= 0.4) return 'yellow'
  return 'red'
}

function getEmotionLabel(flags) {
  if (!flags) return '—'
  if (flags.conflict) return '冲突'
  if (flags.passive) return '消极'
  if (flags.anxious) return '焦虑'
  return '积极'
}

function getEmotionDesc(flags) {
  if (!flags) return ''
  const issues = []
  if (flags.conflict) issues.push('冲突')
  if (flags.passive) issues.push('消极情绪')
  if (flags.anxious) issues.push('焦虑')
  return issues.length > 0 ? `⚠ 检测到${issues.join('、')}` : '情绪健康'
}

function hasNegativeEmotion(flags) {
  return flags?.conflict || flags?.passive || flags?.anxious
}
</script>

<template>
  <div class="grid grid-cols-4 gap-4">
    <div
      v-for="card in cards"
      :key="card.title"
      class="bg-white rounded-2xl border border-gray-200 p-5"
    >
      <div class="flex items-center justify-between mb-3">
```

### 步骤 4：参与度环形图

**文件：** `frontend/src/components/dashboard/ParticipationChart.vue`

```vue
<script setup>
import { computed } from 'vue'
import { Doughnut } from 'vue-chartjs'
import {
  Chart as ChartJS, ArcElement, Tooltip, Legend
} from 'chart.js'

ChartJS.register(ArcElement, Tooltip, Legend)

const props = defineProps({ snapshots: Array })

const latestSnapshot = computed(() => props.snapshots?.[0])

const chartData = computed(() => {
  const scores = latestSnapshot.value?.participation_scores || {}
  const labels = Object.keys(scores)
  const values = labels.map(l => parseFloat((scores[l] * 100).toFixed(1)))
  
  const colors = ['#3b82f6', '#ef4444', '#10b981', '#f59e0b', '#8b5cf6']
  
  return {
    labels,
    datasets: [{
      data: values,
      backgroundColor: colors.slice(0, labels.length),
      borderWidth: 0,
      hoverOffset: 4
    }]
  }
})

const chartOptions = {
  responsive: true,
  plugins: {
    legend: { position: 'right', labels: { font: { size: 12 } } },
    tooltip: {
      callbacks: {
        label: (ctx) => `${ctx.label}: ${ctx.parsed}%`
      }
    }
  }
}
</script>

<template>
  <div class="bg-white rounded-2xl border border-gray-200 p-5">
    <h3 class="font-semibold text-gray-800 mb-4">参与度分布</h3>
    
    <div v-if="latestSnapshot" class="flex items-center justify-center h-48">
      <Doughnut :data="chartData" :options="chartOptions" />
    </div>
    <div v-else class="h-48 flex items-center justify-center text-gray-400 text-sm">
      暂无数据
    </div>
  </div>
</template>
```

```vue
<script setup>
import { computed } from 'vue'
import { Line } from 'vue-chartjs'
import {
  Chart as ChartJS, CategoryScale, LinearScale,
  PointElement, LineElement, Tooltip, Legend
} from 'chart.js'

ChartJS.register(CategoryScale, LinearScale, PointElement, LineElement, Tooltip, Legend)

const props = defineProps({ snapshots: Array })

const chartData = computed(() => {
  // 按时间升序（snapshots 默认降序，需反转）
  const sorted = [...(props.snapshots || [])].reverse()
  
  const labels = sorted.map(s =>
    new Date(s.created_at).toLocaleTimeString('zh-CN', { hour: '2-digit', minute: '2-digit' })
  )
  
  return {
    labels,
    datasets: [
      {
        label: '论点多样性',
        data: sorted.map(s => (s.diversity_score || 0) * 100),
        borderColor: '#3b82f6',
        tension: 0.3,
        fill: false
      },
      {
        label: '协作均衡性',
        data: sorted.map(s => (s.balance_score || 0) * 100),
        borderColor: '#10b981',
        tension: 0.3,
        fill: false
      },
      {
        label: '任务进展',
        data: sorted.map(s => (s.progress_score || 0) * 100),
        borderColor: '#f59e0b',
        tension: 0.3,
        fill: false
      }
    ]
  }
})

const chartOptions = {
  responsive: true,
  scales: {
    y: { min: 0, max: 100, ticks: { callback: v => v + '%' } }
  },
  plugins: {
    legend: { position: 'top' }
  }
}
</script>

<template>
  <div class="bg-white rounded-2xl border border-gray-200 p-5">
    <h3 class="font-semibold text-gray-800 mb-4">协作指标趋势</h3>
```

### 步骤 9：分析数据 API 封装

```javascript
import api from './api'

export const analyticsApi = {
  async getSnapshots(roomId, params = {}) {
    return api.get(`/analytics/rooms/${roomId}/snapshots`, { params })
  },
  
  async getInterventions(roomId, params = {}) {
    return api.get(`/analytics/rooms/${roomId}/interventions`, { params })
  },
  
  async getSummary(roomId) {
    return api.get(`/analytics/rooms/${roomId}/summary`)
```

## 五、关键技术细节

### 5.1 教师 WebSocket 连接鉴权

教师连接与学生连接使用**同一 WebSocket 端点**（`/ws/{room_id}`），但后端 `ConnectionManager` 在注册连接时记录用户角色：

```python
# manager.py
self._rooms[room_id].append({
    "ws": websocket,
    "user_id": str(user.id),
    "role": user.role.value  # "student" or "teacher"
})
```

`analysis:update` 事件通过 `broadcast_to_teachers()` 只向 `role=teacher` 的连接发送，学生端不接收此事件（不暴露 AI 调度逻辑）。

### 5.2 Chart.js 响应式更新

使用 `vue-chartjs` 时，`data` prop 变化自动触发图表重绘。通过 Vue 的 `computed` 响应式属性连接 `latestSnapshot`，WebSocket 推送更新后图表自动刷新，无需手动调用 `chart.update()`。

### 5.3 实时状态展示策略

| 数据类型 | 更新方式 | 频率 |
|----------|----------|------|
| 4 个指标卡 | `analysis:update` WebSocket 推送 | 每次分析完成（约 5 分钟） |
| 参与度环形图 | 同上 | 同上 |
| 5 维度雷达图 | 同上 | 同上 |
| 时间轴折线图 | 快照历史累积 | 实时追加 |
| 干预记录 | `agent:stream_end` 后轮询 | 每次 AI 发言 |
| 消息监控流 | `chat:new_message` WebSocket | 实时 |
